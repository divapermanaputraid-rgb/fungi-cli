# FungiCode Project Memory

## Product

FungiCode is an open-source multi-provider AI coding terminal. The public CLI command is `fungi`.

## Current Direction

- Build the core CLI first.
- Use TypeScript + Node.js 20+ + pnpm.
- Support 9Router, OpenAI-compatible adapter, Gemini, and DeepSeek in MVP.
- Use model profiles: fast, smart, coder, planner, reviewer.
- Default permission mode is ask.
- Keep project memory local.
